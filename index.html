Aggiungi una voce

Compila i campi e clicca "Genera voce": in basso apparirà un blocco di testo già formattato. Copialo e incollalo dentro l'elenco ITEMS in cima al file index.html (vedi istruzioni), poi carica di nuovo il file su GitHub.

Questo sito è un unico file HTML: non c'è un database, quindi le nuove voci vanno aggiunte modificando il file stesso. Sotto trovi il testo già pronto da incollare.

Clicca sul punto della mappa per impostare la posizione.

Carica il file .gpx dell'escursione: verrà mostrato in anteprima qui sotto e la traccia comparirà sulla mappa della sezione Escursioni. Dislivello e posizione di partenza vengono impostati automaticamente dai dati del file.
